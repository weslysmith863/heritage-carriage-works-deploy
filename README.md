# Heritage Carriage Works

Inventory and build management web application for Heritage Carriage Works,
a custom horse-drawn carriage shop based in Lexington, Kentucky.

Built with Java 17, Spring Boot 2.6.6, Spring Security, Spring Data JPA,
Thymeleaf, and MySQL. Deployed on Render.com with a Railway-hosted database.

## Running Locally

Requires Java 17, Maven, and a running MySQL instance.
Configure database credentials in application.properties before running.

```
./mvnw spring-boot:run
```
